# Reaction Buttons

The `<cn-reaction-button>` element creates a clickable input for adding a social reaction (like, love, etc.) to a post or comment etc. 
It supports all the nouns that a `<cn-icon>` supports (as it uses `<cn-icon>` internally).

## Examples

import CnReactionStates from '../../components/elements/CnReactionStates.md';

<div class="grid two-col">
  <div class="light border border-radius p-1 surface">
    <h3>LIGHT</h3>

  </div>
  <div class="dark border border-radius p-1 surface">
    <h3>DARK</h3>

  </div>
</div>

## Usage

### Properties


| Property | Description | Default |
|------|-------------|---------|
| `ariaPressed` | Aria pressed attribute, syncs with checked | 'false' |
| `checked` | Checked state, syncs with ariaPressed | false |
| `count` | Number of reactions, only visible if 0 or more | -1 |
| `noun` | Noun for the reaction | 'love' |
| `small` | Small size | false |
| `inactive` | Inactive state, does not react to clicks (for own entries, where the user can view the reactions) | false |
